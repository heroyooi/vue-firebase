# vue-firebase

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

### 실습
```command
npm i @vue/cli
vue create .
npm run serve
```

### 확장 프로그램
- Vetur
- ESLint
- Git History
- indent-rainbow
- Auto Close Tag
- Auto Complete Tag
- Auto Rename Tag

#### EsLint 관련 세팅
```Json
{
  "editor.codeActionsOnSave": {
    "source.fixAll": true
  }
}
    
```
